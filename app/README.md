# [ngn13.fun](https://ngn13.fun) | my personal website
This is the frontend application for my personal website, it's written in SvelteKit and vanilla CSS

## development setup
```bash
git clone https://github.com/ngn13/ngn13.fun && cd ngn13.fun
npm i
npm run dev
```

## build setup
```bash
git clone https://github.com/ngn13/ngn13.fun && cd ngn13.fun
npm i
npm run build
```
